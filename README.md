          <li><b>ll_clear()</b> — clear the list (keep it) and free all nodes</li>
          <li><b>ll_destroy</b> — completely free memory and delete linked list container</li>
          <li><b>ll_insert_at_head()</b> — insert a node with the defined value at the head of the list</li>
          <li><b>ll_insert_at_tail()</b> — insert a node with the defined value at the tail of the list</li>
          <li><b>ll_insert_after()</b> — insert a node with the defined value after the node with specified value</li>
